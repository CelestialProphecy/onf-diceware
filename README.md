## Installation

`npm install https://github.com/btoll/diceware.git -g`

## Usage

    Command | Description
    ------------ | -------------
    generate | Generate a passphrase using the Diceware method.

    Option | Description
    ------------ | -------------
    -n=NUM | The length of the passphrase (defaults to six words).
    -h, --help | Display help.
